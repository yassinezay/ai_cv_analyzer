## Quick Start

### 1. Install dependencies

```bash
pip install -r requirements.txt
python -m nltk.downloader stopwords punkt
```

### 2. (Optional) Download the Kaggle dataset

The notebook works with **synthetic data** by default.
For real results, download the Kaggle Resume Dataset:

```bash
# Option A – Kaggle CLI
pip install kaggle
kaggle datasets download -d snehaanbhawal/resume-dataset
unzip resume-dataset.zip
mv Resume.csv data/resumes.csv

# Option B – Manual download
# https://www.kaggle.com/datasets/snehaanbhawal/resume-dataset
# Save as: data/resumes.csv
```

### 3. Run the notebook

```bash
jupyter notebook notebooks/project_analysis.ipynb
```

The notebook will:
- Load and explore the dataset
- Train the TF-IDF + Logistic Regression baseline
- Train a Bidirectional LSTM with 3 hyperparameter experiments
- Plot training curves and overfitting analysis
- Evaluate both models (accuracy, F1, confusion matrix)
- Save trained models to `data/models/`

### 4. Launch the Streamlit demo

```bash
streamlit run app/streamlit_app.py
```

Then open http://localhost:8501 in your browser.

## Models

### Baseline: TF-IDF + Logistic Regression
- Fast (< 5 seconds)
- Strong interpretability
- Serves as performance lower bound

### Deep Learning: Bidirectional LSTM
```
Embedding → Dropout → BiLSTM (2 layers) → Linear → CrossEntropy
```
- Captures word order and contextual patterns
- Trained in < 5 minutes on CPU
- 3 experiments: hidden dim {64, 128} x dropout {0.3, 0.5}

### Pretrained (bonus): DistilBERT Feature Extractor
- Frozen DistilBERT encoder + trainable classifier head
- Enable `RUN_DISTILBERT=True` in the notebook

## Training Constraints (CPU compliant)

All models respect the **< 10 min CPU** training constraint:
- Max 2,000 training samples (stratified subset)
- Sequence length capped at 150 tokens
- Vocabulary limited to 8,000 words
- 5 epochs max per experiment
